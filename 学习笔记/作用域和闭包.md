# 实际开发中闭包的应用场景，举例说明
- 用于隐藏数据，只提供API
```js
function createCache(){
    const data={};
    return {
        set: function(key,value){
            data[key]=value;
        },
        get: function(key){
            return data[key];
        }
    }
}
let c = createCache();
c.set('a',100);
console.log(c.get('a'));//100
```

知识点：
1. 作用域和自由变量

2. 闭包
自由变量的查找是在函数定义的地方，向上级作用域进行查找，而不是在执行的地方
3. this
this取值，在函数执行时确认，而不是在函数定义时确认
